# Lab 2 - Plan a virtual team-building event with Microsoft 365 Copilot Chat

Your manager has asked you to organize a virtual team-building event for your team. You’ll use **Microsoft 365 Copilot Chat integrated into M365 apps** to brainstorm ideas, organize notes, create a budget, draft documents, send invitations, and generate a PowerPoint deck. You'll learn how Copilot Chat uses the context of your open files and prompts to streamline workflow cross apps. 

The tasks in this lab will guide you on how to navigate and prompt in Microsoft 365 Copilot Chat. To fully complete this lab, you need a Microsoft 365 subscription.

> [!NOTE]
> These tasks are designed specifically for use with **web mode** in Microsoft 365 Copilot Chat. If you have a Microsoft 365 Copilot license, make sure you **manually switch to web mode** when you open Copilot Chat, as it may default to work mode. Using **web mode** ensures the prompts behave as intended and source information from public web content.

## Before you start

Before you can start this lab, you'll need to log into your virtual machine and launch the Microsoft 365 Copilot Chat web app. Follow the steps below to get started: 

1. Log into the virtual machine provided by your tenant provider as the local **Administrator** account with the password `Pa55w.rd`.
   
1. In the Windows taskbar, select **Microsoft Edge**.
   
1. To launch Microsoft 365 Copilot Chat, enter `https://m365copilot.com` in the address bar and a dialogue box will prompt you to sign in.
   
1. At the **Sign-in Prompt**, enter userx@yourtenant.onmicrosoft.com (provided by your tenant provider) and select **Next**.
   
1. At the **Enter password** screen, enter Password1 (provided by your tenant provider), then select **Sign in**..
   
1. If prompted to **Stay signed in?**, select the **Don't show this again** checkbox, then select **Yes**.
   
1. If prompted to **Unlock Copilot Chat in Microsoft 365 Apps - Add Copilot Chat to Microsoft 365 Copilot app, Teams and Outlook for easy access**, feel free to select **Skip for now** or **Add Copilot**, as either action does not have any effect on this lab exercise.
   
1. The Microsoft 365 Copilot Chat web app should launch, if not, select the **Copilot icon** on the left navigation menu to open Copilot Chat.

You are ready to begin Task 1. 

## Task 1: Copilot Chat (web mode) – Brainstorm event ideas

In this task, you’ll start in Copilot Chat (web mode) to brainstorm ideas for your virtual team-building event.

1. Enter this prompt in the prompt box:

   I’m planning a virtual team-building event for our remote team to boost collaboration and morale. Summarize 8 trending ideas for virtual team-building activities in bullet points, with brief descriptions and why they work well for remote teams. Make it concise and professional.

1. Select **Send (arrow icon)** on bottom right of the prompt box or select **Enter** on your keyboard. 

1. Review the information provided by Copilot and if needed, refine the prompt.

1. Enter this follow-up prompt: 

    I’d like to host [virtual team-building event]. Can you suggest and compare 3 [virtual team-building event] providers, including pricing plans for each? Present this in a clear table format.

   **NOTE:** Replace [virtual team-building event] with your chosen event. 

1. Select **Send** or select **Enter**. 

1. Review the information provided by Copilot and if needed, refine the prompt.

## Task 2: OneNote – Organize planning notes

In this task, you'll use Copilot Chat in OneNote to organize your planning notes into actionable tasks to help guide you in your event planning process.

1. Open OneNote on your device and create a new page. 

1. Select the **Copilot icon** at the top right corner of the ribbon and Copilot will open in a side pane.  

1. Open the same chat conversation with Copilot located in the navigation panel under "chats."

1. In the same chat with Copilot, enter this prompt in the prompt box:

    Help me create a to-do list for planning [virtual team-building event], formatted for OneNote. Include tasks like [booking the provider, sending invites, and preparing materials.]
    
    **NOTE:** Replace [virtual team-building event] with your chosen event and replace [booking the provider, sending invites, and preparing materials] with tasks you need to complete.

1. Select **Send** or select **Enter**.  

1. Review the information provided by Copilot and if needed, refine the prompt. 

1. Once you are happy with Copilot's response, copy and paste the information onto your OneNote page. 

1. Enter this follow-up prompt: 

    Prioritize these tasks based on what needs to be completed first.

1. Select **Send** or select **Enter**. 

1. Review the information provided by Copilot and if needed, refine the prompt.

## Task 3: Excel – Build and analyze the budget

In this task, you'll use Copilot Chat in Excel to build and analyze an estimated budget for the event and create a visual representation of the budget. 

1. Navigate back to the Copilot Chat app.

1. In the same chat with Copilot, enter this prompt in the prompt box:

    Help me build an estimated budget using [virtual event provider] as the host for this virtual team-building event. Assume 30 attendees and include prize costs for winners. Create this as an Excel file.
     
    **NOTE:** Replace [virtual event provider] with your chosen provider. 

1. Select **Send** or select **Enter**. 

1. Review the information provided by Copilot and if needed, refine the prompt.

1. Download and open the Excel file Copilot Chat created and save it. 

1. Select the **Copilot icon** at the top right corner of the ribbon and Copilot will open in a side pane. 

1. Open the same chat conversation located in the navigation panel under "chats."

1. Enter this prompt: 
    
    Summarize the cost breakdown for each pricing scenario for leadership review. Include a visual chart.

1. Select **Send** or select **Enter**. 

1. Review the information provided by Copilot and if needed, refine the prompt.

1. Add the content into your Excel sheet and save.

    Help me create a fun graphic to include in the email invite. The graphic should showcase [blue and green colors, collaboration, happy people, and technology].

1. Select **Send** or select **Enter**. 

1. Review the information provided by Copilot and if needed, refine the prompt.

1. Add the content into your email and save.
